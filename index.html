Projects from BSc

BSc Project Laboratory

Project Laboratory is the first individual project work in the BSc level. My project was about learning the basics of IoT systems, design and implement a prototype version of a weather station. Avaliable measurements: temperature, ambient light level, atmospheric pressure. The measuring system communicates with a central unit with wireless technology: Bluetooth.
Weather station IoT Wireless communication Bluetooth Low power consumption Raspberry Pi Arduino Uno Sensors Ambient light Temperature Humidity Barometric pressure

BSc Thesis

My thesis was an advanced version of my weather station. It was about improving the previous project to a more complex IoT system. The embedded hardware was improved, and I implemented a web server with real-time database for storing and sharing the measured data and a graphical client side web application for displaying the data from my weather station.
